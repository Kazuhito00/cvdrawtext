# cvdrawtext
cvdrawtextはOpenCVでフォントを指定して文字を描画するクラスです。

# Requirement
 
* OpenCV 3.4.2
* Pillow 6.1.0
 
# Installation
 
利用したいPythonプログラムと同階層にcvdrawtextディレクトリをコピーしてください。
 
# Usage
 
サンプルの実行方法は以下です。
フォントは同梱していないため、お試しの際は後述のURLからダウンロードして格納してください。
 
```bash
python sample.py
```
